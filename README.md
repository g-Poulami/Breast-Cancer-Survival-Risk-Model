# Breast Cancer Survival & Risk Modelling
This project implements a clinical risk stratification model using the METABRIC dataset.

## Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Run the notebook in `notebooks/analysis.ipynb`

## Methodology
- Data: METABRIC (cBioPortal)
- Survival Analysis: Kaplan-Meier Estimator
- Predictive Modeling: Cox Proportional Hazards
- Performance: C-index and Calibration Plots
